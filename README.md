# Mac环境变量

* 配置GOPATH
* 配置GOBIN

```
vim ~/.bash_profile
export GOPATH="your workspace path"
export GOBIN="your workspace path/bin"
export PATH=$PATH:$GOPATH/bin

source ~/.bash_profile
```
